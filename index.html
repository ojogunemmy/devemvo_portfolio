⚓ Port Management Association of West and Central Africa

Developed a comprehensive web platform for the Port Management Association of West and Central Africa (PMAWCA), an organization established in 1972 under the auspices of UNECA. The platform serves 24 regular member ports and 8 associate members spanning from Mauritania to Angola along the West African coast, handling over 300 million tons of maritime import/export trade across a 12,000 km coastline.

Built with MongoDB, Express, React, and Node.js (MERN stack), the platform facilitates knowledge sharing and cooperation on best practices in port management and operations. Features include multilingual support for English, French, and Portuguese to serve Anglophone, Francophone, and Lusophone countries, member directories, resource libraries, and collaboration tools for delivering efficient port services while maintaining safe, secure, and environmentally clean operations.

Key Impact: Enabling seamless communication and knowledge exchange across 40+ member organizations, contributing to improved quality of port services and promoting sustainable socio-economic development across the West and Central African sub-region.

Engineered an intelligent content automation system that transforms keywords into fully published, SEO-optimized WordPress articles with zero manual intervention. Key Impact: Reduces content production time by 70-80%, generates SEO-optimized articles with proper structure (H2/H3/H4 headings, bullet points, tables, FAQs), supports batch processing via TXT/CSV inputs, and enables seamless WordPress integration with automatic categorization, tagging, and featured image uploads transforming content creation from hours to minutes.

🎮 LibGDX OpenAPI Client Generator

Developed a custom OpenAPI code generator specifically tailored for LibGDX game framework, producing Java code that's fully compatible with Google Web Toolkit (GWT), desktop, and mobile platforms. The generator creates API client libraries using LibGDX's native networking stack instead of incompatible libraries like OkHttp or Retrofit.

Built using OpenAPI Generator with custom Mustache templates and CodeGen extensions, the tool generates API classes that utilize com.badlogic.gdx.Net.HttpRequest for network requests and com.badlogic.gdx.utils.Json for serialization/deserialization. Includes CLI integration and Gradle task support for seamless build pipeline integration, ensuring all generated code is Java 8 compatible and works across GWT/HTML, desktop, and mobile builds without modification.

Key Impact: Eliminates manual API client coding for LibGDX projects, ensures cross-platform compatibility out of the box, reduces integration time from days to minutes, and provides reusable template system for future API specifications enabling game developers to focus on game logic rather than networking boilerplate.

Java LibGDX OpenAPI Generator GWT Mustache Templates Cross-Platform
